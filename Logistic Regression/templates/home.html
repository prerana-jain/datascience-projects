Description:

The goal of this project is to develop a logistic regression model that can predict whether the customer is satisfied or not based on various factors such as Inflight entertainment, Baggage handling, Cleanliness, Departure, Departure Delay, Arrival Delay, etc. The model will be trained on a dataset of historical weather data and will be evaluated based on its accuracy in predicting the weather.

Logistic regression is a type of regression analysis used for predicting the outcome of a categorical dependent variable, based on one or more predictor variables. It is a widely used machine learning algorithm in data science and statistics.
